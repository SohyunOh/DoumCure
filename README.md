## 주제에 따른 결론

### 입력

데이터 요청시 필요 데이터 파라미터 호출을 통해 데이터 요청

- 필요데이터 : 제품명, 효능(증상), 사용법(연령별)

### 처리

불러온 데이터를 게시판 형식으로 나열하여 보여주는 방식

동기화 처리 방식으로 예상함. 

프로젝트 진행상황에 따라 변동 가능성 있음

### 결과

나온 결과를 문자 그대로 출력 or 문자별 이미지매칭(map)을 통한 이미지 출력 논의 예정

## 부주제에 따른 결론
